# DjangoApp

This repository contains two separate projects:

1. **Bootstrap Frontend Exercises** — Simple HTML/CSS/Bootstrap practice projects.  
2. **Django Book App** (Lab4) — A Django application for managing books (Create, Read, Update, Delete).

## Features

- Basic Django setup with models, views, and templates  
- URL routing  
- Simple CRUD operations (Create, View, Update, Delete)  
- Uses Django’s built-in admin panel  
- Basic UI components (navbar, cards, containers, etc.)
